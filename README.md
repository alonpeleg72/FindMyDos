# FindMyDos - מעקב אחר הפגנות חרדיות בישראל

FindMyDos היא יישום לסריקת מקורות מידע (אתרי חדשות, רשתות חברתיות ומקורות רשמיים) כדי לזהות הפגנות של ציבור חרדי בישראל. כאשר מזוהה הפגנה, המערכת מציגה את המיקום, בודקת האם היא משפיעה על כבישים ראשיים (כמו כביש 4, כביש 6 וכו'), ומספקת קישור למקור יחד עם מפה שמראה את המיקום.

## תכונות

- סריקה אוטומטית של מקורות מידע למציאת הפגנות חרדיות
- זיהוי שפה עברית בטקסט
- זיהוי הפגנות וקשר לציבור חרדי
- חילוץ מיקום מטקסט וגיסוק קואורדינטות
- ניתוח השפעה על כבישים ראשיים
- מאגר נתונים לאחסון הפגנות מזוהות
- ממשק אינטרנטי להצגת הפגנות עם מפות אינטראקטיביות
- סריקה תקופתית לעדכון אוטומטי
- מניעת כפילויות בזיהוי הפגנות

## דרישות מוקדמות

- Python 3.9+
- pip (מתקין חבילות Python)
- חיבור לאינטרנט (לסריקת מקורות ולגיסוק מיקומים)

## התקנה

1. שכפל את המאגר או העתק את קבצי היישום לספרייה על המחשב שלך

2. צור סביבת וירטואלית (מומלץ):
   ```bash
   python -m venv venv
   source venv/bin/activate  # במערכת Windows: venv\Scripts\activate
   ```

3. התקן את החבילות הנדרשות:
   ```bash
   pip install -r requirements.txt
   ```

4. העתק את קובץ דוגמת משתני הסביבה והגדר אותו לפי הצרכים שלך:
   ```bash
   cp .env.example .env
   ```
   ערוך את קובץ `.env` והוסף את הערכים הנדרשים

5. אתחל את מאגר הנתונים:
   ```bash
   flask db upgrade
   ```
   או הפעל את היישום פעם אחת כדי ליצור את הטבלאות אוטומטית

## שימוש

כדי להפעיל את היישום במצב פיתוח:
```bash
flask run
```
או
```bash
python run.py
```

היישום יהיה זמין בכתובת: http://localhost:5000

לסריקה ידנית של מקורות מידע (ללא שימוש במתזמן):
```bash
python -c "
from findmydos.scheduler import UpdateScheduler
scheduler = UpdateScheduler()
scheduler._perform_update()
"
```

## הגדרה

משתני סביבה שניתן להגדיר בקובץ `.env`:

- `FLASK_ENV`: סביבת הפעלה (development, production, testing)
- `SECRET_KEY`: מפתח סודי ליישום Flask (חובה בסביבת ייצור)
- `DATABASE_URL`: כתובת חיבור למסד נתונים (ברירת מחדל: SQLite בקובץ מקומי)
- `SCRAPING_INTERVAL_HOURS`: intervals בין סריקות במקורות מידע (ברירת מחדל: 2 שעות)
- `USER_AGENT`: מחרוזת User-Agent לבקשות HTTP
- `REQUEST_TIMEOUT`: זמן ממתנה לבקשות HTTP בשניות

## כיצד זה עובד

1. **סריקה**: היישום סורק אתרי חדשות ישראליים נבחרים (כמו Ynet ו-Haaretz) לתוכן חדש
2. **זיהוי שפה**: הטקסט שנאסף עובר זיהוי שפה כדי למקד בטקסט עברי
3. **זיהוי הפגנה**: הטקסט נבדק לנוכחות מילים קשורות להפגנות ולמחאות
4. **זיהוי ציבור חרדי**: הטקסט נבדק לנוכחות מזהים לציבור חרדי/חסידי
5. **חילוץ מיקום**: משתמשים בזיהוי ישויות מוגדרות (NER) ובסיס נתונים של ערים ישראליות ידועות כדי לחלץ מיקומים
6. **גיסוק מיקום**: שירות גיסוק חיצוני (Nominatim/OpenStreetMap) ממיר שמות מיקומים לקואורדינטות
7. **ניתוח השפעה על כבישים**: בודקים אם ההפגנה קרובה לכבישים ראשיים ידועים או אם הטקסט מזכיר השפעה על תנועה
8. **אחסון**: המידע נשמר במאגר נתונים עם מנגנון למניעת כפילויות
9. **הצגה**: הממשק האינטרנטי מציג את ההפגנות עם פרטים, קישור למקור ומפה אינטראקטיבית
